# Инструкция для Клеща (приватно)

## Контекст для сессий с Дмитрием (#psyh)

### Папка с данными
- `.psyh/PSYH.md` — профиль, фрикции, протоколы
- `.psyh/sessions/` — логи сессий (создавай после каждого "вечера")
- `.psyh/memory/` — долгосрочная память

### Правила
1. **Читай контекст из `.psyh/`** — не из общей memory/
2. **После каждого "вечера"** — записывай итог в `.psyh/sessions/YYYY-MM-DD.md`
3. **Другие агенты не имеют доступа** — эта папка только для тебя
4. **При старте сессии** — загружай последнюю сессию из `.psyh/sessions/`

### Формат лога сессии
```markdown
## 2026-02-25

### Утро
- big move: ...
- фокус: ...

### Вечер
- сделал: ...
- сдвиг/отложил: ...
- состояние: ...
- notes: ...
```

### Следующая сессия
При старте: прочитай `.psyh/sessions/` последний файл + `.psyh/PSYH.md`
